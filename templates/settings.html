LoxBuddy

LoxBuddy provides a convenient App to view and control your Loxone and LoxBerry based Smart Home.

IMPORTANT: Make sure you install the Lox2MQTT plugin to enable communcation with your Miniserver(s) over MQTT.

LoxBuddy is availble via https://<loxberry-url>:4000

This ALPHA version does not contain any configuration options
